About This Project

Offline Encryption & Decryption Web Application

This project is developed as part of the BSc Computer Science curriculum. It demonstrates the basic concepts of cryptography such as encryption, decryption, encoding, and hashing using a web-based interface.

The application works completely offline and performs all operations on the client side using JavaScript. No data is sent to any server, ensuring privacy and security.

The project helps users understand how encryption algorithms protect sensitive information in real-world applications such as secure communication and data storage.
