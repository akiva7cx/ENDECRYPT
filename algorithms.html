Algorithms Used

1. AES (Symmetric Encryption)

AES is a symmetric key encryption technique where the same secret key is used for both encryption and decryption. In this project, AES is demonstrated using XOR-based logic for academic understanding.

2. Base64 Encoding

Base64 is an encoding method that converts binary data into readable ASCII characters. It is commonly used in email and web applications.

3. SHA-256 Hashing

SHA-256 is a cryptographic hash function that converts input data into a fixed-length hash value. Hashing is one-way and cannot be reversed.

Why Client-Side?

All algorithms are executed on the client side to ensure offline functionality and data privacy.
